About the Project

This is a test version of TWRP Recovery specifically compiled for the Samsung Galaxy A13 (SM-A135M). The project aims to provide a stable custom recovery solution for this device.

Installation Guide

  1. Download the recovery image
  2. Make sure you have USB debugging enabled
  3. Install the recovery using Odin or Fastboot
  4. Boot into recovery mode to verify installation
